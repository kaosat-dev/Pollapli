Pollapli by kaosat-dev

Parent project for the Doboz-web and Hydroduino projects (migration in progress): general tool for remote monitoring and control of projects (arduino, reprap, hydroponics etc)

Dependencies

python2.6/2.7, twisted, pyserial

Install

for now the project is using zc.buildout for deployment -get the files from github -in the same folder type python bootstrap.py - type buildout/buildout -then launch using buildout/pollapli
