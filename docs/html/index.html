Welcome to Doboz-web’s documentation!¶

Warning

Currently (version 0.5.0) Pollapli is not even an alpha version, features and documentation are partially implemented and being reworked a lot, so use at your own risk!

Warning

Don’t be surprised if you see both the names “Pollapli” and “DobozWeb” , Doboz Web is actually going to be part of the larger application called Pollapli, so in the transitionning phase , a lot of text will be bear reference to both!

Are you a user?¶

Pollapli was thought from the start with end-users (that is you !) in mind, making it as easy to use as possible, without too much technical knowledge.
